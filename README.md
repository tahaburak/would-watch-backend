# Would Watch - Backend

Go (Golang) API server for the Would Watch application.

## Tech Stack
-   **Language**: Go 1.25+
-   **Data**: Supabase (PostgreSQL)
-   **External APIs**: TMDB (The Movie Database), OpenAI

## Setup
1.  Copy `.env.example` to `.env`.
2.  Run `go mod download`.
3.  Run `go run cmd/api/main.go`.
